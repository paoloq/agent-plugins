# code

Claude Code plugin with skills for assessing and improving source code.

## Skills

- **review** — Read-only assessment of a repository (or a subfolder) that surfaces a prioritized list of pain points: bugs, security risks, anti-patterns, complexity hotspots, refactoring opportunities, dead/duplicated code, missing tests, dependency smells, performance concerns, and documentation gaps. Invoke as `/code:review [path]`.

## Install

Add this plugin's directory to your Claude Code plugins.
